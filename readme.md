# This application was created to manage your daily tasks
## Used technologies

- Tailwind CSS
- TypeScript
- OOP oriented programming
- Drag and Drop API
## Run Locally

Clone the project

```bash
  git clone https://github.com/Fiji404/My-Todos.git
```

Go to the project directory

```bash
  cd My-Todos
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev-start
```
