## Run Locally

Clone the project

```bash
  git clone https://github.com/Fiji404/MyTodos.git
```

Go to the project directory

```bash
  cd MyTodos
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
